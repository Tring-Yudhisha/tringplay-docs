# Authentication & User Management

## 4.1 Supported Authentication Types

TringPlay supports multiple authentication providers:

- **Cognito-based authentication**
- **Custom Authentication APIs**
- **Cleeng Authentication**
- **InPlayer Authentication**

Authentication can be:

- Enabled or disabled globally
- Configured per platform
- Switched dynamically via configuration
