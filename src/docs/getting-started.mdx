# Getting Started

## Introduction

**TringPlay** is a white-label OTT streaming product, similar in capability to platforms like Netflix and Zee TV. It is designed to be **backend-agnostic**, highly configurable, and scalable across multiple platforms, enabling clients to integrate TringPlay apps with their own backend systems while retaining full control over authentication, subscriptions, content, and features.

The primary goal of TringPlay is:

- To allow **API-level flexibility**, so apps continue to work even if the backend changes
- To enable **feature toggling and configuration changes** without rebuilding the app
- To provide **multi-platform OTT delivery** from a single configurable product

## Supported Platforms

TringPlay supports a single product architecture across:

- Web
- Android Mobile
- iOS Mobile
- Android TV
- Apple tvOS
- Amazon Fire TV
- Roku
- Samsung Smart TV
- LG Smart TV
- Vizio Smart TV

Each platform supports independent:

- **Feature enablement**
- **Player configuration** (default or custom)
- **Independent UI and behavioral settings**

## Client Value Proposition

TringPlay enables clients to:

- **Launch OTT apps faster**
- **Change backend systems** without app rewrites
- **Enable or disable features instantly**
- **Customize branding and UI**
- **Scale across platforms seamlessly**
